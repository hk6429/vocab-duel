# 字鬥英雄 vocab-duel

從國小 1200 字到國中 2000 字的英文單字練功站——閃卡、單字自測、學習戰績，之後還會加入文學家對戰、字首字尾心智圖、會考考古題。

## 功能（P1）

- **閃卡練功**：Leitner 五盒間隔複習（0/1/2/4/7 天），到期優先、新字補位，一回合 20 張
- **單字自測**：英選中／中選英／例句挖空 三題型隨機，四選一、同級誘答，一輪 10 題，結果回寫熟悉度
- **我的戰績**：已掌握單字（第 3 盒以上）、學段進度條、今日複習數、連續天數
- **免帳號**：進度存 localStorage，可用進度碼匯出／匯入跨裝置搬移

## 字庫

教育部「參考字彙表」常用 2000 字詞（含國中小最基本 1200 字詞標記），雙來源交叉驗證解析，中譯、詞性、例句自產並經抽樣校對。資料在 `data/words.json`，原始清單與解析報告在 `data/raw/`。

## 開發

純靜態站，無框架、無建置步驟。

```bash
# 本機預覽
python3 -m http.server 8080

# 資料驗證
node scripts/validate.mjs

# 煙霧測試（需 playwright-core）
node scripts/smoke.mjs
```

## 路線圖

- P1 ✅ 字表 + 閃卡 + 自測 + 統計
- P1.5 高中詞彙表（大考中心 Level 1–6）
- P2 對戰模式：電腦對手（莎士比亞、海明威等 8 位古典文學家）＋同機雙人
- P3 字首字尾拆解 + 心智圖
- P4 會考英文考古題（錯題自動進閃卡）
- P5 線上即時對戰
