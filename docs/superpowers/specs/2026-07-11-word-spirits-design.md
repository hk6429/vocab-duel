# 詞靈系統（Word Spirits）設計文件

日期：2026-07-11　狀態：已由大乃核准（方案 A＋區塊 1-5 全做）

## 一句話

把字綴關聯圖與寵物養成對戰綁成一個閉環：**每隻寵物守護一組字綴，你學會該家族的字，寵物就變強**——看圖等於看軍團實力，養寵慾直接驅動學單字。

## 核心決策（已確認）

| 決策 | 選擇 |
|---|---|
| 落點 | 整合進字鬥英雄（vocab-duel），沿用字庫/字幣/XP/Upstash |
| 結合機制 | 方案 A：字綴親和——20 隻詞靈各綁一組字綴，詞源之力=該家族已學字比例 |
| PvP | 非同步影子對戰（雲端快照，AI 代打對方寵物） |
| 美術 | 水彩墨風生圖，每寵 3 階段進化圖（共 60 張），裝備/裝飾用 emoji+CSS |

## 系統設計

### 1. 詞源星圖（新頁 graph）
- Canvas 力導向圖：20 詞靈節點（外環）→ 172 字綴節點，節點大小/亮度 ∝ 熟悉度
- 點詞靈或字綴 → 展開該家族單字子圖（已學亮、掌握金亮、未學灰）
- 側欄統計：總字量、最熟 Top5 字綴（%）、最弱 Top5（＝練功建議，點了直接跳閃卡）
- 6205 字不全量入模擬——主圖只跑 20+172 節點，單字子圖按需展開

### 2. 詞靈養成（新頁 pets）
- 20 隻詞靈：4 字首系、8 字尾系、8 字根系（172 綴不重複全分完，驗證腳本把關）
- 屬性：`atk = (10 + 2×Lv) × (1 + P) + 裝備atk`；`hp = 100 + 6×Lv + 裝備hp`；P=詞源之力（家族已學字/家族總字）
- 等級 1–25，升級花字幣 `cost(L→L+1) = 20 + 10L`；進化：Lv1 幼年 / Lv10 成熟 / Lv25 完全體（換圖+慶祝）
- 技能 3 槽：Lv5/Lv12/Lv20 解鎖（共用技能池，每寵預設組合）：連擊增傷、受擊護體、背水爆發、開場先手、吸血、詞源共鳴（P>60% 額外增傷）
- 裝備 4 槽（武器/護甲/飾品/紋章）：對戰勝利掉落，稀有度沿用開箱三檔（普通/稀有/傳說），敵人越強掉落檔位越高
- 裝飾：緞帶/花冠/圍巾/眼鏡等 CSS 疊加；特寫鏡頭：放大主圖+呼吸動效+潑墨背景
- 首隻詞靈免費領養，其餘用字幣解鎖（前 5 隻 100 幣、之後遞增）——字幣新大出口

### 3. 詞靈對戰（pets 頁內入口）
- 答題驅動：答對=寵物出招（傷害吃 atk+技能），答錯=敵方反擊；沿用 VDQuiz 出題、與文學家對戰同手感
- PvE 野生梯度：10 層野生詞靈（強度遞增，掉落遞好）
- 影子 PvP：快照上傳 Upstash（`vd:pet:` 前綴），挑戰隨機對手快照，AI 依對方強度代打；寵物積分勝+20 敗−10（獨立於文學家段位）
- 寵物排行榜：全站 Top50（`vd:petboard:global`），比戰力/積分/詞源之力

### 4. 經濟閉環
背單字賺字幣（現有）→ 領養/升級詞靈（新出口）→ 想變強得學該家族字綴的字（回學習）→ 對戰贏裝備+幣 → 循環。現有商店/寶箱/每日任務/段位全保留互通；VDGame.onAnswer 不動，寵物層掛在外圍。

### 5. 工程切分
| 檔案 | 職責 |
|---|---|
| `data/pets.json` | 20 詞靈定義＋172 字綴分組＋技能池＋野生梯度 |
| `js/petstore.js` | 寵物狀態(localStorage `vd_pets`)、詞源之力、屬性/經濟計算、裝備、純邏輯零 DOM |
| `js/petgraph.js` | 詞源星圖 Canvas＋統計側欄 |
| `js/pet.js` | 養成 UI（清單/詳情/升級/裝備/裝飾/特寫） |
| `js/petbattle.js` | 對戰（PvE＋影子 PvP） |
| `api/pets.js` | 快照上傳/抽對手/排行榜（Vercel serverless，CF/Netlify 前端指回 vercel.app，同 cloud.js 模式） |
