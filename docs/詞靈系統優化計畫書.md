# 字鬥英雄・詞靈（寵物）系統優化計畫書

> 2026-07-18 ｜ 依 10 人網路遊戲設計專業團隊評估綜整
> 對象：大乃老師（作者）｜前提：完全免費・零內購・純前端＋CF D1・國中小・繁中台灣・白帽八角

---

## 一、一句話結論

詞靈系統有一顆**世界級的心臟**——「詞源之力」（戰力＝家族已學字比例，想變強就得去學牠的字根字綴），這是全台學習遊戲罕見的正確白帽耦合、是護城河。**任何優化都不能動它的骨架，只能加深。**

但這顆心臟目前有兩處**致命出血**，都只需改公式常數／計分方式（純前端、零金流）即可止血：

1. **「碰過就算已學」（box≥0）門檻太低** → 學生會刷已學衝戰力、而非真精熟，把殺手鐧偷偷變成打卡。
2. **裝備量級輾壓學習乘數**（13 階 ×1.8，頂裝約 1157 倍 vs 詞源之力 ×2 上限，且裝備加在乘數外面）→ 越後期、玩最兇時，戰力越由「刷裝」決定，學習耦合被架空。

**優化總目標＝把耦合做深、收斂贅肉、用「學習深度」而非「數值長尾」解斷崖。** 不是再加系統，是把對的地方鎖死、把偷渡的黑帽拆掉。

## 二、現況體檢

### 該保留、不能動的殺手鐧（10 位專家全員共識）
- **詞源之力耦合**：攻擊力 = (10+2×等級) × (1+詞源之力) + 裝備；家族學越多、牠越強。系統北極星。
- **升級硬門檻**：「詞源之力% ≥ 等級×3」不學字就升不上去，用機制強制耦合，比說教有效。
- **徹底白帽的乾淨經濟**：字幣唯一來源＝答對題與任務、不能買、零內購零廣告。家長／學校信任的地基。
- **把 172 個抽象字綴人格化成 20 隻有臉的神獸**——很好的認知鷹架，把「構詞法」變成有情感容器的使命感。
- **60 張水彩潑墨立繪**：免費專案裡的奢侈品，與文人鬥法／文豪笑傳美術語言一致、有跨作品綜效。

## 三、分階路線圖

> 每項標「影響力／成本／教育契合度」。**P1＝先做（止血＋救公式＋去黑帽，全部低成本高保護）**。

### P1｜止血與正骨（建議立刻做，幾乎都是改公式／文案，純前端）

**1. 詞源之力改 Leitner 盒級加權，堵死「刷已學」後門** 〔影響高／成本低〕
把「已學比例」從 box≥0 的 0/1，改成依複習盒盒號加權的家族平均（例：box0=0.2、box2=0.6、box3+=1.0）。攻擊力與升級門檻沿用同一加權值；UI 把戰力拆兩條顯示「學過的量 vs 精熟的深度」。
→ *10 人一致點名、投報率最高的一改。用最小改動讓戰力從「可刷的假精熟」升級成「真精熟」。*

**2. 救攻擊力公式：裝備收進乘數＋壓平裝備係數** 〔影響高／成本低〕
公式由 `atk = base×(1+詞源) + 裝備` 改成 `atk = (base+裝備)×(1+詞源)`（讓裝備也吃詞源乘數）；裝備每階倍率 ×1.8 → 約 ×1.3、階數 13 → 4~5 階重算。可再設軟上限（裝備加成 ≤ 學習項某比例）。
→ *確保任何裝備等級下「學得多」永遠壓過「運氣好」，把「學習永遠是戰力主體、裝備只是糖霜」寫進數學結構。*

**3. 競技去黑帽：敗不扣分、排行榜改成長導向** 〔影響高／成本低〕
移除「敗-10」，積分只增或每週重置；全站淨勝榜改成「個人成長曲線＋本季精熟成長幅度＋週最佳」；配對依實力／年級分池。
→ *扣分與全站榜踩白帽紅線、對低成就學生殺傷最大（英雄島已踩過同雷）。改成「跟昨天的自己比」。*

**4. 鍛造去賭博化：N 件同階「必成」1 高階** 〔影響中／成本低〕
移除機率與失敗吞料，改確定性合成（集滿必升）或保底計數；想保留驚喜只做「額外詞條有機率更好」這種只賺不賠的正向隨機。
→ *拆掉家長最敏感的「教孩子賭」觀感雷。*

**5. 立一句話北極星＋攻擊力可視化拆解，常駐 UI** 〔影響高／成本低〕
定調「**你的詞靈有多強，取決於你真的懂多少構詞——牠是你單字腦的化身**」，每隻立繪旁常駐詞源之力儀表＋這句話；攻擊力做成可展開的來源拆解條（基礎 X＋詞源加成 Y＋裝備 Z）。
→ *把價值敘事從裝備跑分手上搶回來，讓「養詞靈＝練單字腦」每一屏都看得到。*

### P2｜解斷崖、補回訪與擁有感

- **6. 每日「到期字綴任務」**〔高／中〕：每天派 1~2 個「今日主打字綴家族」（優先挑今天到期複習或詞源偏低的），完成到期複習＝餵養對應詞靈，給當日限定小額字幣＋24h 增益；溫和共學日曆（斷了不倒扣）。*同時解「無每日鉤子」＋「複習盒與遊戲脫節」。*
- **7. 詞靈「精通位階／家族大師」**〔高／中〕：Lv25＋詞源 100% 後開 ★1~★5，升階條件是「維持精熟」（整組養到 box3+／連續 N 週不掉盒），視覺用既有立繪疊星星光環 overlay（CSS 即可），給榮譽不硬塞戰力。*零新美術補住 D30 斷崖，獎勵「長期不遺忘」。*
- **8. 詞靈取名＋羈絆台詞＋可分享名片**〔高／中〕：自訂每隻名字、升級/進化/連對跳主題台詞、canvas 生成可截圖「詞靈名片」（立繪＋名字＋等級＋詞源%＋圖鑑完成度＋約戰房號）。*情感綁定第一槓桿＋「不能課金但想炫耀」的唯一出口＝免費招生。*
- **9. 技能改可自選 loadout**〔中／中〕：保留 6 技池與家族身份，改成解鎖後自由配 3 個上陣、可少量重洗；順手把「背水怒濤（殘血加傷）」改成「連對達 X 次觸發」以移除獎勵答錯的反誘因。
- **10. 對戰題型差異化（改考拆字綴/構詞）**〔中／高〕：主線維持記字義；詞靈對戰改成該家族的構詞題（拆積木、由字根猜生詞、組字），拆得完整觸發暴擊。*「輸入 vs 輸出」分工，把戰鬥從換皮測驗升級成構詞應用。建議先小範圍或當必殺技觸發，別一次全量換掉熟悉手感。*

### P3｜長線與大膽

- **11. 融合鬆綁＋重定位為「跨家族構詞洞察」**〔中／中〕：移除雙親降級懲罰、門檻下修 Lv15、做成圖鑑收集；幼靈誕生給「兩組字綴共享的語源故事」與跨家族構詞連結圖。
- **12. 字綴挑戰季／主題週 LiveOps**〔高／中〕：每 1~2 週輪播主題（「-tion 名詞週」「否定字首對決週」），當季家族複習/對戰給額外獎勵與限定戳章，軟榜以「本季精熟成長幅度」計名。*一張 config 表就能上新一季，零美術炒熱既有 172 字綴。*
- **13. 漸進解鎖子系統＋首頁「今天的一件事」導引卡**〔中／中〕：以進度門檻分層開放（裝備第 3 隻後、融合/星圖 Lv25 才亮）；首頁固定一張大卡＋單一 CTA，前端算出此刻最該做的動作（「阿墨詞源 42%，去學 3 個 -tion 家族的字就能升 Lv7」直接跳學習清單）。
- **14. 產出型學習：造詞／傳承銘文**〔中／中〕：精通詞靈解鎖「傳承」——為家族的字自造例句/口訣/合法構詞，通過前端規則校驗後成為圖鑑「銘文」、可匿名成別人的提示卡（輕量 UGC，D1 一張表）。*把最投入的學生從「輸入」推到「產出、教別人」的 Bloom 最高階。*

## 四、三個最大膽、值得認真賭的點子

1. **「活體肖像／記憶生命體」**：立繪不再是固定三階，而是由真實家族精熟數據即時驅動的「看得見的大腦」——精熟越高靈紋越亮、氣場越盛（純前端 SVG/CSS 濾鏡疊在既有水彩上，零新美術）；字久未複習掉盒則顯「想你了」並列出最想再見的 3 個字，點下去直接進複習。**把間隔複習的自然衰退，轉成白帽的正向召喚（餵養而非扣血）。**
2. **「親手造詞生詞靈」**：把 20 隻固定收集升級成無限自造——用已精熟的字綴（un＋break＋able）親手拼新詞靈，白名單詞庫＋規則校驗判合法構詞，幼靈名字＝造出來的字。**一刀同解斷崖（追求無限）＋炫耀（人人獨一無二）＋課金替代（不花錢也有「只有我有」），把學生從消費者變生產者。**
3. **雙貨幣「字髓」把終局戰力鎖進真學習**：字幣當日常軟通貨，另開稀有硬通貨「字髓」，唯一產出是把字推進最高盒或通過字綴大師挑戰，買不到、刷簡單題也刷不到；最終進化／傳說詞條／限定融合只收字髓。**讓刷已學、刷簡單題、刷裝三條套利路都只能買到中段戰力，經濟結構本身成為白帽守門員。**

## 五、風險守則（優化時務必守住）

1. **凡不耦合學習的系統都在跟殺手鐧搶注意力**。加任何新玩法前先問：「這會讓學字更被看見，還是把注意力導向刷數值？」
2. **盒級加權會讓現有玩家戰力當場回落**——必須用正向框架溝通（「你的詞靈餓了，回去複習餵養牠會更強」），**絕不能做成扣血／倒數威脅**，否則自己踩進黑帽。
3. **砍裝備階數／鍛造會讓重度玩家覺得內容變薄**——務必同步用精通位階、主題季、造詞等「學習深度」長尾補上，**別只砍不補**。
4. **「延遲回憶才算餵養」這類最硬的科學正確，可能過度嚴苛打擊每日熱情**——當天仍給基礎回饋，把「隔日精熟」設為額外加成而非唯一。
5. **白帽社交炫耀與社會比較只有一線之隔**——展示一律比「個人成長／精熟成果」而非「絕對戰力／淨勝」，配對分池保護後段班。
6. **系統收斂要漸進**——用「漸進解鎖」把複雜度往後藏，別直接刪除玩家已擁有的內容。

## 六、建議的開工順序

**先做 P1 五項（quick wins）**——全部低成本、純前端改公式／文案，卻能一次救回整套系統的教育誠信：

1. 詞源之力改盒級加權（堵後門）
2. 攻擊力公式把裝備收進乘數（救輾壓）
3. 競技敗場不扣分＋排行榜改成長導向（去黑帽）
4. 鍛造改「N 件必成」（去賭博）
5. 北極星金句＋攻擊力來源拆解條常駐 UI（把價值講清楚）

P1 落地、驗證數值與體感後，再進 P2（每日鉤子＋精通位階＋取名名片）解斷崖與回訪。P3 與三個大膽賭注（尤其「活體肖像」「親手造詞」）等 P1/P2 站穩再評估。
